# CAPABILITIES 1 - FileCleaner

## PUEDE HACER
- Subir CSV/XLSX autenticado y procesarlo en background via `SystemOutbox`.
- Limpiar filas vacias, duplicados exactos y whitespace.
- Ejecutar magic-clean para emails, telefonos, precios, fechas y nombres.
- Detectar duplicados fuzzy bajo demanda.
- Exportar metadata de archivos procesados.
- Ofrecer demo publica limitada.

## NO PUEDE HACER
- Cumplir aun con metadata stripping, compresion de imagenes o conversion de formatos prometidos en landing.
- Mostrar resumen agregado de calidad/ahorro.
- Hacer preview/dry-run sin storage.
- Procesar archivos grandes de forma streaming para todas las rutas.

## LIMITACION RAIZ
- Tecnica y de posicionamiento: la implementacion real evoluciono hacia data cleaning, pero el marketing/config aun describe file utility generico.

## IMPACTO
- Alto: el producto tiene valor real, pero la promesa externa puede atraer usuarios equivocados y generar churn.
