# CAPABILITIES 5 - FeedbackLens

## PUEDE HACER
- Crear feedback manual.
- Importar feedback por texto bulk o CSV.
- Analizar sentimiento con Gemini y fallback local.
- Detectar urgencia.
- Generar borrador de respuesta.
- Guardar preferencias de prompt, alert email y weekly digest.
- Crear summary semanal y enviar digest.
- Exportar feedback.

## NO PUEDE HACER
- Mostrar el weekly summary en dashboard como decision layer.
- Comparar tendencias contra periodos previos.
- Integrarse con fuentes externas.
- Validar limite de prompt personalizado.
- Deduplicar feedback similar.

## LIMITACION RAIZ
- Diseno/producto: la API tiene mas valor que el dashboard actual expone.

## IMPACTO
- Alto: con un panel de insights visible, el producto comunica mucho mejor el valor de AI.
